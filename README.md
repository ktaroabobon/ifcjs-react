# ifcjs-react

Sample project to show how to use [IFC.js](https://ifcjs.github.io/info/) in a React app.

## Usage

clone

```bash
git clone https://github.com/ktaroabobon/ifcjs-react.git
```

setup

```bash
make setup
```

install dependencies

```bash
make install
```

preview

```bash
make start/serve
```
